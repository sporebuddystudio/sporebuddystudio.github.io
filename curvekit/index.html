Installation

1. Import CurveKit into your Unity project via the Asset Store.

2. Open Unity and load your scene.

3. Select a GameObject that contains a Unity Spline.

4. Activate the CurveKit overlay tool.

CurveKit works directly with Unity Splines. No additional setup is required.

Getting Started

After activation, CurveKit allows you to edit splines using advanced editing operations directly inside the Scene View.

  • Round corners
  • Insert new points
  • Append segments
  • Cut splines
